# Fil-Am Foodie

## Description 
Fil-Am Foodie is a recipe blog featuring Filipino and American dishes. Users can view recipes by category and leave comments and replies on each recipe page.

* React 
* JavaScript 
* Node.js 
* HTML
* CSS
